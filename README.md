# 🧮 Calculator Python

> Простой, но функциональный калькулятор на Python с историей вычислений

![Python](https://img.shields.io/badge/python-3.7%2B-blue?style=flat-square)
![License](https://img.shields.io/badge/license-MIT-green?style=flat-square)

---

## ✨ Возможности

- ➕ **Сложение** — складывает два числа
- ➖ **Вычитание** — вычитает второе число из первого
- ✖️ **Умножение** — умножает два числа
- ➗ **Деление** — делит первое число на второе (с защитой от деления на 0)
- 🔋 **Возведение в степен��** — возводит число в степень
- 📜 **История вычислений** — отслеживает все выполненные операции
- ⚡ **Интерактивный режим** — удобный командный интерфейс

---

## 🚀 Быстрый старт

### Требования
- Python 3.7 или выше

### Использование

```bash
python Calculator.py
```

### Пример работы

```
Калькулятор запущен. Доступные операции: +, -, *, /, **!!
Для выхода введите 'exit', для истории введите '/history'.

Введите первое число (или exit): 10
Введите операцию (или /history): +
Введите второе число: 5
Результат: 15.0

Введите первое число (или exit): /history
---История вычислений ---
10.0 + 5.0 = 15.0

Введите первое число (или exit): exit
```

## 🔧 API Класса Calculator

### Методы операций

```python
from Calculator import Calculator

calc = Calculator()

# Прямое использование методов
calc.add(10, 5)          # → 15
calc.substract(10, 3)    # → 7
calc.multiply(4, 5)      # → 20
calc.divide(10, 2)       # → 5.0
calc.power(2, 3)         # → 8
```

### История

```python
calc.show_history()  # Выводит все проведённые операции
```
